## 什么是 Clash Verge？

Clash Verge 是一款备受用户青睐的开源 Clash 图形用户界面 (GUI) 客户端，专为 Windows、macOS 和 Linux 用户设计。它在强大的 Clash 核心功能之上，提供了一个现代化、直观且易于操作的界面。如果您正在寻找一个能够帮助您轻松管理网络代理规则、切换代理节点，并优化网络连接体验的工具，那么 Clash Verge 无疑是一个值得尝试的选择。它不仅简化了复杂的配置过程，还通过友好的用户体验，让即使是初次接触代理工具的用户也能快速上手，享受流畅的网络冲浪。

## 核心功能概览

Clash Verge 拥有一系列强大而实用的功能，旨在提升用户的代理管理效率和体验。首先，它支持多种配置文件的导入方式，无论是订阅链接还是本地 YAML 文件，都能轻松导入并进行管理。用户可以根据需求快速切换不同的配置档案。其次，Clash Verge 提供了灵活的代理模式选择，包括全局代理、规则代理和直连模式，让您能够精确控制哪些流量通过代理，哪些直连。此外，实时的节点延迟测试功能可以帮助您选择最佳的代理节点，确保网络速度和稳定性。界面方面，它支持深色和浅色主题，并允许用户进行个性化定制。内置的系统代理管理功能则让您一键开启或关闭系统代理，无需手动设置，极大地提升了便捷性。

## 如何开始使用 Clash Verge？

开始使用 Clash Verge 的过程非常简单。首先，您需要从官方渠道下载并安装适用于您操作系统的 Clash Verge 客户端。安装完成后，首次启动应用，您会看到一个简洁的用户界面。接下来，最关键的一步是导入您的代理配置文件。通常，这可以通过在“配置文件”或“Profiles”页面粘贴订阅链接并点击“下载”按钮来完成，Clash Verge 会自动拉取并解析您的代理规则。如果您有本地的 YAML 配置文件，也可以通过“导入”功能加载。成功导入配置文件后，您可以在节点列表中选择一个您希望使用的代理节点。最后，在主界面启用“系统代理”开关，Clash Verge 便会接管您的网络流量，并根据您的配置规则开始工作。此时，您就可以畅游互联网，享受更自由、更安全的网络体验了。
