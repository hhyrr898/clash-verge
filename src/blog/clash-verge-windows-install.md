## 安装前准备

在 Windows 设备上使用 Clash Verge，建议先确认系统版本、管理员权限和安全软件提示。安装包保存到固定目录后再运行，可以减少后续查找配置文件时的混乱。

准备阶段重点看三个位置：安装目录、配置保存目录、系统托盘入口。Clash Verge 启动后通常会在托盘区显示图标，用户可以从这里打开面板、切换连接状态或退出程序。

## 安装流程

打开下载中心后，选择 Windows 版入口，确认安装包名称和设备系统一致。运行安装程序时，如果系统出现确认窗口，按提示继续即可。安装完成后，建议先从桌面或开始菜单打开一次 Clash Verge，确认主界面能够正常显示。

如果设备上曾安装旧版本，可以先保留原有配置，再进行新版本安装。这样即使界面或入口发生变化，也可以通过原订阅快速恢复使用环境。

## 首次启动检查

首次打开 Clash Verge 后，先查看主界面是否正常显示，再进入配置区域导入订阅或本地文件。导入完成后，建议先进行一次延迟查看，确认列表能够正常刷新。

如果界面没有变化，可以重新打开程序，或检查系统代理是否被其他工具占用。Windows 上常见的冲突来自浏览器插件、旧版代理工具和企业安全策略。

## 连接状态确认

导入配置后，先选择一个常用节点，再查看连接状态。随后打开浏览器访问常用页面，确认系统代理已经生效。若浏览器正常但其他桌面应用异常，可以检查该应用是否使用独立网络设置。

## 日常使用建议

日常使用时，不建议频繁更换太多配置。保持一个主用配置和一个备用配置即可。遇到连接异常时，先切换节点，再查看日志提示，最后再考虑重新导入订阅。
